# AUEB NLP group page

A full redesign of the Natural Language Processing laboratory of the Athens Univeristy of Economics and Business. 
Features a simplistic design, modern HTML/CSS practices and full compliance with [AUEB's design standards](https://www.aueb.gr/el/logotypa).

Requested by the Department Head to eventually replace the [current website](https://nlp.cs.aueb.gr/index.html).
